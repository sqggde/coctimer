> 对应提交: `b0a4897`

## 1. 新增卡片

在**账号信息**与**升级项目列表**之间新增了一个分类概览卡片，横向展示 5 个分类的升级进度：

```
┌──────────────────────────────────────────────┐
│  [🏠]      [🧪]      [🐾]      [🌙]      [🚀]   │
│  ● 5/6     ● 0/1      1/1      ● 2/2      1/1  │
└──────────────────────────────────────────────┘
```

### 分母计算规则

| 分类 | 图标 | 分母 = 以下建筑在数据中的 cnt 总和 |
|---|---|---|
| 主世界建筑 | `1000015_1.png` | `1000015`(工人小屋) + `1000064`(小博木屋) |
| 实验室 | `1000007_16.png` | `1000007`(实验室) |
| 战宠 | `1000068_11.png` | `1000068`(战宠小屋) |
| 夜世界建筑 | `1000047.png` | `1000034` + `1000047`(博仔棚屋) + `1000078` |
| 星空实验室 | `1000046_10.png` | `1000046`(星空实验室) |

- 分子 = 该分类**正在升级**的项目数量
- 分母 = 对应建筑在 JSON 中的 `cnt` 累加和（无 cnt 字段时按 1 算）

## 2. 红点提醒

当分子 < 分母时，对应分类的文字左侧会出现红色 `●` 提示。

**控制方式：**

| 操作 | 效果 |
|---|---|
| **点击红点文字**（如 `● 5/6`） | 弹出屏蔽选项 |
| → **单次屏蔽** | 当前页面不显示该分类红点，下次导入 JSON 后恢复 |
| → **永久屏蔽** | 存入 `settings.dismissedCategories`，永久隐藏红点 |
| → **取消** | 什么都不做 |
| **长按图标 3 秒** | 弹出取消屏蔽确认框（仅当该分类已被屏蔽时有效） |
| → **确认取消** | 清除单次 + 永久屏蔽记录，红点恢复显示 |

### 技术实现

- 红点显隐用 `style.opacity = '0'/'1'` 控制，**不触发 DOM 重排**，解决闪烁问题
- 点击和长按交互在 `switchAccount` 时一次性绑定，不在每秒的定时器里重复绑定
- 单次屏蔽：内存变量 `sessionDismissedCategories`（导入时 `sessionDismissedCategories = {}` 清空）
- 永久屏蔽：`settings.dismissedCategories` → `localStorage`
