# 部落小工具 - COC Timer Small Tool

一个《部落冲突》(Clash of Clans) 升级进度追踪工具。

## 功能

- 导入游戏数据，自动解析正在升级的建筑物、实验室、战宠等
- 实时显示剩余时间，支持工人助手、实验室助手等加速效果
- 多账号管理，支持备注名
- 夜间模式
- 宝箱刷新提醒
- 账号排序

## 使用方法

1. 在游戏中导出 JSON 数据
2. 点击「粘贴导入」粘贴 JSON
3. 自动显示所有升级项目及剩余时间
